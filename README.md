## ✅ Topics Covered

### 1. Arrays & Hashing
Basic array operations, frequency counting, hash maps, duplicate detection, and pair-sum problems.

### 2. Strings
String traversal, palindrome logic, anagrams, substring problems, and pattern-based questions.

### 3. Linked List
Pointer manipulation, reversing lists, cycle detection, merging lists, and advanced linked list patterns.

### 4. Stack & Queue
Parentheses problems, monotonic stack, next greater element, daily temperatures, and queue-based logic.

### 5. Binary Search
Classic binary search, search space reduction, rotated arrays, and answer-based binary search.

### 6. Recursion
Base cases, recursive calls, call stack understanding, and recursive problem solving.

### 7. Backtracking
Subsets, permutations, combinations, N-Queens, Sudoku, and decision-tree based problems.

### 8. Trees & BST
Tree traversals, height/depth problems, BST validation, LCA, and tree construction.

### 9. Graphs
DFS, BFS, islands, connected components, topological sort, and shortest path basics.

### 10. Dynamic Programming
1D DP, 2D DP, subsequence DP, grid DP, and optimization-based problems.
